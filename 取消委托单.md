
### 1.9 `cancel` 取消委托

/api/v1/cancel

**描述**：

此接口用来取消委托

**请求参数**


| 参数名          | 类型     | 是否必须 | 描述   |
| :----------- | :----- | :--- | :--- |
| accessKey | String | 是    | AccessKey |
| market | String | 是    | 市场 |
| id | long | 是    | 订单ID |
| singnature | String | 是    | 签名后的字符串 |

**返回结果**

```
{
	"code": 200,
	"info": "success."
}
```
